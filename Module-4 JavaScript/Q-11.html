What are all the looping structures in JavaScript?

  • If we want to run same code multiple times with differt result than loops are used.
  • There are five typs of loop in java script.
    1. while
    2. Do/while
    3. for
    4. for in
    5. for of
  • While, for, for in and for of are entry control loop while do\while is exit control loop.
  • Example
    for(i=1; i<=5; i++)
    {
    document.write("<br>")
    for(j=1; j<=i; j++)
