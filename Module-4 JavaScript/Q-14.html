What are all the types of Pop up boxes available in JavaScript

  • Three types of pop up box avaliable in java script.
    1. alert()
    2. Confirm()
    3. Prompt()
  • alert(); :-> This pop up box comes with one button and return only true value.
  • Confirm(); :-> This pop up box comes with ok and cancle button and return true and false value.
  • prompt(); :-> This pop up box comes with two button and one input value from user.
  • Exmple:
    function alrt(){
    alert("Heloo.....")
    }
    function Confirm(){
    confirm("Press ok or cancle..")
    }
    function promt(){
    prompt("What's your favorite cocktail drink?");
    }
