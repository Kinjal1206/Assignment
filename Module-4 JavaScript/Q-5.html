What are undeclared and undefined variable?

  • Undeclared variable means that variable which is used in program without declaraction using key word let, const and var.

  • Example

    function aa(){
    x.style.color="blue";
    }
    In above example "x" is undeclared variable.

  • Undefined variable means variable is declared with keyword let, const and var but not assign any value to it.

  • Example

    function aa(){
    var a = 10;
    var b = 5;
    var c;
    }
    In above example variable "c" is undefined variable.
