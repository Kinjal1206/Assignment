Q7. How to install react js in windows linux operating system? How to install npm and how to check version of npm?

  • Before installing react we will require to install node.js.
  • Download leatest version of node.js andinstall it.
  • open commend prompt and just enter commend:
    npx create-react-app myapp
    It is create myapp application.
  • The node js is contains npm so with current installation of node js, npm has installed too.
  • To check version of npm you can write commend
    -> npm -v
