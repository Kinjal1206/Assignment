4) Name a few CSS style components:


    CSS consists of two components:
  • Properties:
    Properties are use to modify style features.
    Example, font-size , width , background-color.
  • Values: Each property is assigned a value. This value indicates how to style the property.
    Example: Font-size:"10px"

  • Example:
    "color: blue"
    "font-size: 25px"
    This is blue color Text with 25px font size.
